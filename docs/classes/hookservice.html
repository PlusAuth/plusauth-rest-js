addPackages

  • addPackages(hookId: string, packages: HookPackage[], stream: true): Promise<any>
  • addPackages(hookId: string, packages: HookPackage[], stream: undefined | false): Promise<string>
    • Defined in src/api/hooks.ts:121

    Add npm packages to hook. Added packages will use exact versions to prevent inconsistencies in the future. Even you pass 'latest' as version, the resolved version will be the latest exact version at that time.

    example
    const output = await plusAuth.hooks.addPackages('HOOK_ID', [ { name: 'axios', version: '0.19.1' } ])
    console.log(output) // prints whole npm install output
    example

    Stream response

    const stream = await plusAuth.hooks.addPackages('HOOK_ID', [ { name: 'axios', version: '0.19.1' } ], true)
    stream.on('data', function( data ){
      console.log(data)
    })
    stream.on('end', function(){
      console.log('END')
    })

    Parameters

    • hookId: string

      Id of the hook to add packages into

    • packages: HookPackage[]

      Array containing packages

    • stream: true

      If true the response will be a stream, streaming the output of install command

    Returns Promise<any>

    • Defined in src/api/hooks.ts:125

    Parameters

    • hookId: string
    • packages: HookPackage[]
    • stream: undefined | false

    Returns Promise<string>
